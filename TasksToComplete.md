# TasksToComplete

## Các phần còn thiếu trong Investor Package

### 1. Financial Model + Unit Economics (LTV, CAC, Payback)
- Chưa có file Excel chi tiết (ví dụ: day18_financial_model.xlsx)
- Chưa có nội dung mô tả chi tiết các chỉ số tài chính, LTV, CAC, Payback
- Task:
  - Hoàn thiện file Excel tài chính (theo hướng dẫn Day 18)
  - Viết mô tả ngắn về mô hình tài chính, các giả định chính, và kết quả tính toán

### 2. Roadmap Now/Next/Later + OKRs (1 quý)
- Chưa có file hoặc nội dung roadmap, OKRs
- Task:
  - Xây dựng bảng Roadmap (Now/Next/Later) cho sản phẩm
  - Viết OKRs cho 1 quý đầu tiên (3-5 mục tiêu, mỗi mục tiêu 2-3 kết quả then chốt)

### 3. Dependency Map + Critical Path
- Chưa có file hoặc nội dung về Dependency Map, Critical Path
- Task:
  - Vẽ sơ đồ hoặc mô tả các phụ thuộc chính (kỹ thuật, dữ liệu, con người)
  - Xác định Critical Path (chuỗi công việc quan trọng nhất ảnh hưởng tiến độ)

### 4. Phụ lục
- Chưa có link file Excel chi tiết Day 18
- Chưa có file PRD đầy đủ Day 17 (chỉ có bản skeleton draft)
- Task:
  - Hoàn thiện file PRD chi tiết (Day 17)
  - Đính kèm hoặc dẫn link file Excel tài chính

### 5. Các mục khác cần kiểm tra
- Đảm bảo các phần Market Analysis, Customer, Pain, Solution đã đủ chi tiết (tham khảo các file Handbook nếu có)
- Task:
  - Đọc lại các file có tên chứa "Handbook" để bổ sung insight, ví dụ: định nghĩa chỉ số, hướng dẫn điền bảng, checklist nộp bài
